# 🎮 オセロゲーム (Othello/Reversi)

HTML/JavaScriptで実装された完全なオセロゲームです。ブラウザで直接プレイできます！

## ✨ 特徴

- 🤖 **AI対戦** - 初級・中級・上級の3段階難易度
- 📊 **AI思考ログ** - AIの思考過程をリアルタイム表示
- 🔄 **戻る機能** - 1手戻すことが可能
- 📱 **レスポンシブデザイン** - スマートフォン対応
- 🎯 **完全実装** - パス判定、ゲーム終了判定など完全なルール
- 🎨 **美しいUI** - モダンなデザイン

## 🎲 遊び方

1. **難易度を選択** - 初級・中級・上級から選べます
2. **「新しいゲーム」をクリック**
3. **黄色いヒントが表示された場所をクリック**
4. **AIと交互にプレイ**
5. **AIの思考ログを見ながら戦略を学ぼう！**

## 🚀 プレイ方法

ブラウザで `オセロゲーム.html` を開くだけでプレイできます！

```bash
# ローカルで開く場合
start オセロゲーム.html
```

または、GitHub Pagesで公開されている場合は直接アクセスしてください。

## 🛠️ 技術仕様

- **言語**: HTML5, CSS3, JavaScript (ES6+)
- **AIアルゴリズム**: ミニマックス法 + 評価関数
- **難易度調整**: 探索深さによる調整
- **UIフレームワーク**: 純粋なCSS (Flexbox/Grid)

## 📈 AIの思考について

AIは以下の要素を考慮して最適な手を選択します：

- **ひっくり返す石の数**
- **コーナー/エッジの位置ボーナス**
- **相手の次の手数を制限**

思考ログを見れば、AIがどのように考えているかがわかります！

## 🎯 ゲームルール

1. 黒(あなた)が先手、白(AI)が後手
2. 相手の石を挟んで自分の石を置くと、挟まれた石が自分の色に変わる
3. 置ける場所がない場合はパス
4. 両者とも置けなくなったらゲーム終了
5. 石の多い方が勝ち

## 📝 開発者向け

### ファイル構成
```
オセロゲーム.html  # メインゲームファイル
```

### 主な関数
- `initBoard()` - ボード初期化
- `getValidMoves(player)` - 有効な手の取得
- `getAIMove()` - AIの手の決定
- `placeAt(x, y, player)` - 石の配置とひっくり返し
- `drawBoard()` - ボードの描画
